Terms below show how an element fits into the document flow

Inline Level Elements:

- <a>
- <img>
- inline elements fit alongside other elements

Block Level Elements:

- <h1> to <h6>
- <p>
- elements that take up a whole "BLOCK" of space

* Inspect element can show you HOW MUCH SPACE an element takes up
-------------------------------------------------------------------------

Generic Containers:

<span>
- inline element

<div> (The Content Division Element)
- block element
- generic container 
- used to group elements together
- forces content above and below it
- used to apply CSS styles to a group of elements
